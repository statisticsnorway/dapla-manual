Felles oversikt over Dapla-relaterte systemer og verktøy

Denne oversikten er ment som et oppslagsverk for Dapla-relaterte fellessystemer, tjenester og verktøy i SSB. Den skal gi en kort og overordnet forklaring av navn som ofte brukes i arbeid med statistikkproduksjon, forskning og forvaltning av data. Oversikten utfyller Godkjentlisten ved å gi korte forklaringer av hva ulike verktøy brukes til, hvem de er relevante for, og hvor man finner mer informasjon. Den erstatter ikke Godkjentlisten og skal ikke forstås som en selvstendig godkjenning av teknologier eller verktøy. Ved valg av verktøy til statistikkproduksjon gjelder Godkjentlisten og øvrige interne retningslinjer.

Hvis noe er feil eller har mangler, ta kontakt med mfm[at]ssb.no.

Sist oppdatert: 25. juni 2026

systemer = FileAttachment("fellessystemer_dapla_hovedoversikt_tags.csv").csv({typed: false})
notBlank = (x) => x !== undefined && x !== null && String(x).trim() !== ""

normaliser = (x) => String(x ?? "")
  .toLowerCase()
  .normalize("NFD")
  .replace(/[\u0300-\u036f]/g, "")
  .replace(/æ/g, "ae")
  .replace(/ø/g, "o")
  .replace(/å/g, "a")

sorterNorsk = (a, b) => String(a).localeCompare(String(b), "nb-NO")

parseTags = (x) => String(x ?? "")
  .split(";")
  .map(t => t.trim())
  .filter(notBlank)

parseTagFilter = (x) => normaliser(x)
  .split(/[\s,;+]+/)
  .map(t => t.trim())
  .filter(notBlank)

systemerMedTags = systemer.map(d => ({
  ...d,
  _tags: parseTags(d.Tags),
  _tagsNormalisert: parseTags(d.Tags).map(normaliser)
}))

alleTags = Array.from(
  new Set(systemerMedTags.flatMap(d => d._tags))
).sort(sorterNorsk)
viewof filtre = Inputs.form({
  sok: Inputs.text({
    label: "Søk",
    placeholder: "Søk etter navn, målgruppe, forklaring eller tag"
  }),
  tags: Inputs.text({
    label: "Filtrer på tags",
    placeholder: "f.eks. metadata + python"
  })
})

Søkefeltet søker bredt i hele oppføringen. Skriver du for eksempel metadata + python, vises bare oppføringer som har begge taggene.

html`<details class="system-tag-details">
  <summary>Vis tilgjengelige tags</summary>
  <div class="system-tag-cloud">
    ${alleTags.map(t => html`<span class="system-tag">${t}</span>`)}
  </div>
</details>`
filtrerteSystemer = systemerMedTags.filter(d => {
  const q = normaliser(filtre.sok)
  const tagFilter = parseTagFilter(filtre.tags)

  const sokefelt = [
    d.Navn,
    d["Målgruppe"],
    d.Forklaring,
    d.Tags
  ].map(normaliser).join(" ")

  const riktigSok = q === "" || sokefelt.includes(q)

  // Alle tags i taggfilteret må finnes. Delvis treff er tillatt,
  // slik at "kontroll" matcher taggen "kontroll-og-editering".
  const riktigTags = tagFilter.length === 0 || tagFilter.every(
    t => d._tagsNormalisert.some(dt => dt === t || dt.includes(t))
  )

  return riktigSok && riktigTags
})
html`<div class="system-results-summary">
  <strong>${filtrerteSystemer.length}</strong> av <strong>${systemer.length}</strong> oppføringer vises.
</div>

${filtrerteSystemer.length === 0
  ? html`<div class="no-results">Ingen treff. Prøv færre tags eller et bredere søk.</div>`
  : html`<div class="system-card-list">
      ${filtrerteSystemer.map(d => {
        const kilde = d["Kilde / mer informasjon"]
        return html`<article class="system-card">
          <h3>${d.Navn}</h3>
          ${d._tags.length ? html`<div class="system-tags">${d._tags.map(t => html`<span class="system-tag">${t}</span>`)}</div>` : ""}
          ${notBlank(d.Forklaring) ? html`<p>${d.Forklaring}</p>` : ""}
          ${notBlank(d["Målgruppe"]) ? html`<p class="system-meta"><strong>Målgruppe:</strong> ${d["Målgruppe"]}</p>` : ""}
          ${notBlank(kilde) ? html`<a class="system-link" href=${kilde} target="_blank" rel="noopener noreferrer">Mer informasjon</a>` : ""}
        </article>`
      })}
    </div>`
}`
